## 📌 About This Repository

This repository contains lecture slides, datasets, problem sets, and Stata code for ECON 645. It is a public resource for enrolled students. Please use [ELMS/Canvas](https://elms.umd.edu) for submitting assignments, accessing grades, and course announcements.

> ⚠️ **Submissions go to ELMS, not GitHub.** This repo is for accessing materials only.
